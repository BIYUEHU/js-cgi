# CGI-JS

Base on Rust and QuickJS implementation of CGI protocol. Let JavaScript can simply run Nginx, similar to traditional PHP running mode.

## Features

**Fast and Lightweight**: Use Rust and QuickJS to implement.
**Easy to Use**: Provides a simple and intuitive API for developers.
**No toolchain**: Applications based on Express/NestJS Framework dependend toolchain and need build to use production, but CGI-JS scripts can run directly.
**Unified**: Client script and server script all written in JavaScript.
